## QUICK START (3 STEPS)

### STEP 1: Get Supabase Credentials
1. Go to https://supabase.com
2. Create a new project (or use existing one)
3. Go to **Settings → API** 
4. Copy:
   - **Project URL**
   - **Anon Key** (public key)

### STEP 2: Update Configuration
1. Open `supabase-config.js`
2. Replace these lines:
   ```javascript
   const SUPABASE_URL = 'YOUR_SUPABASE_URL';
   const SUPABASE_ANON_KEY = 'YOUR_SUPABASE_ANON_KEY';
   ```

Example:
   ```javascript
   const SUPABASE_URL = 'https://yourproject.supabase.co';
   const SUPABASE_ANON_KEY = 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...';
   ```

### STEP 3: Create Database Tables
1. Open **Supabase SQL Editor**
2. Copy all SQL from `wellmeadows_tables_sql.sql`
3. Paste into Supabase SQL Editor
4. Click **Run**

## TROUBLESHOOTING

**Problem: Connection fails**
- Check your SUPABASE_URL and SUPABASE_ANON_KEY are correct
- Make sure Supabase project is active
- Check browser console for error messages (F12)

**Problem: "Table does not exist"**
- Make sure you ran the SQL to create tables
- Check table names are exactly correct (case-sensitive)
- Example: `STAFF` not `staff`, `PATIENT` not `Patient`

**Problem: Can't insert data**
- Check table structure matches your data
- All required fields must be provided
- Check browser console for specific error

**Problem: No data loading**
- Make sure tables are created in Supabase
- Check there's actually data in the tables
- Open browser console (F12) to see error messages
